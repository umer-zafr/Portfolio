Project Overview

Car Low App is a comprehensive car rental and management application built with Flutter. The app provides users with a seamless experience to browse, book, and manage car rentals with real-time location tracking and payment integration.

The application leverages Firebase for backend services including authentication, real-time database, and cloud storage. It integrates Google Maps API for location services, enabling users to find nearby cars and track their rentals in real-time.

Key Features

Firebase Authentication

Secure user login and registration with Firebase

Car Browsing & Search

Browse available cars with advanced filters and search

Real-time Location Tracking

Track car locations and navigate using Google Maps

Booking Management

Easy booking process with calendar integration

Payment Integration

Secure payment processing for rentals

User Profiles

Manage user profiles and rental history
