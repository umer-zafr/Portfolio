Project Overview

Get Live App is a comprehensive food and restaurant management application built with Flutter. This app provides users with a seamless experience to browse, book, and manage food orders and table bookings with Firebase integration.

The application leverages Firebase for backend, including authentication, real-time database, and data storage. Supabase APIs are integrated for advanced data management and user profile packages to deliver a smooth and reliable user experience.

Key Features

Freelance Authentication

Secure user login and registration with Firebase

Car Streaming & Search

Browse cars by date and advanced filter and search

Real-time Data

Manage car data real-time via booking with Supabase API

Booking Management

Track all bookings and calendar integration

Payment Integration

Secure payment processing for orders

User Profiles

Manage user data and saved preferences
