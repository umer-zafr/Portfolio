Project Overview

Quizler App is an interactive quiz application built with Flutter that challenges users with engaging questions across multiple categories. The app features a clean, intuitive interface with real-time score tracking, making learning fun and competitive. Perfect for testing knowledge and improving retention through gamified learning.

Questions fetched from API with multiple categories and difficulty levels

Time-based challenges to add excitement and urgency

Category Selection

Restart & Review
