# Images Folder

This folder contains all images for the portfolio website.

## Recommended Images to Add

### Profile Photo
- **File name:** `profile.jpg` or `profile.png`
- **Recommended size:** 500x500px (square)
- **Format:** JPG or PNG
- **Note:** Update the path in `data/personal.js`

### Project Screenshots
Create project-specific folders or use these naming conventions:
- `project-healthtrack-1.jpg`
- `project-healthtrack-2.jpg`
- `project-smartshop-1.jpg`
- etc.

**Recommended sizes:**
- Main project image: 800x600px
- Detail page screenshots: 1200x800px
- Mobile screenshots: 400x800px

### Optimization Tips
1. Compress images before uploading (use tools like TinyPNG)
2. Use WebP format for better performance
3. Keep file sizes under 500KB
4. Use descriptive file names

## Image Formats Supported
- JPG/JPEG - For photos
- PNG - For graphics with transparency
- WebP - For modern browsers (best performance)
- SVG - For icons and logos

---

**After adding images, update the file paths in:**
- `data/personal.js` (for profile image)
- `data/projects.js` (for project images)
