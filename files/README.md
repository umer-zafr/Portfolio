# Files Folder

This folder contains downloadable files for the portfolio.

## Adding Your Resume

1. Create your resume PDF file
2. Name it: `Umer_Zafar_Resume.pdf`
3. Place it in this folder
4. The download button on the homepage will automatically link to it

## Supported File Types

- PDF (Recommended for resumes)
- DOCX (Alternative resume format)
- Other documents as needed

---

**Note:** Make sure your resume file name matches the one specified in `data/personal.js`
