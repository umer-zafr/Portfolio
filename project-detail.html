Project Overview

A comprehensive e-commerce mobile application built with Flutter, designed to provide a seamless shopping experience for fashion retail customers. The app features a modern, intuitive interface with smooth animations and responsive design that works perfectly on both iOS and Android platforms.

The application includes real-time inventory management, secure payment processing through Stripe integration, user authentication with Firebase, and a robust admin panel for managing products, orders, and customer data.

Key Features

User Authentication

Secure login with email, Google, and Apple Sign-In

Product Catalog

Browse products with filters, search, and categories

Shopping Cart

Add, remove, and manage items with real-time updates

Payment Gateway

Secure payments via Stripe with multiple payment methods

Order Tracking

Real-time order status updates and notifications

Wishlist

Save favorite items for later purchase
