Project Overview

A simple yet elegant BMI (Body Mass Index) Calculator built with Flutter. The app helps users calculate their BMI based on height and weight inputs, providing instant results with health status indicators. The interface features a modern dark theme with intuitive controls and clear visual feedback.

The application provides gender selection, height adjustment via slider, and weight/age selection with increment/decrement buttons. Results are displayed with color-coded health status (Normal, Underweight, Overweight) making it easy for users to understand their BMI category at a glance.

Key Features

Choose between Male and Female options
